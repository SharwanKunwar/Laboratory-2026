## 1. Explain global attributes in HTML.
Global attributes are attributes that can be applied to all HTML elements.

**Examples:**

* id → unique identifier
* class → grouping elements
* style → inline CSS
* title → tooltip


<br><br>

## 2. Explain types of CSS.

1. Inline CSS → inside element
2. Internal CSS → inside style
3. External CSS → separate file

***External CSS is best for large projects.***

<br><br>

## 3. What are block-level elements?

Block-level elements take full width and start on a new line.

**Examples:**

```
<div>
<p>
<h1>
```
<br><br>

## 4. Explain LINK tag.

Used to connect external resources like CSS.

```html
<link rel="stylesheet" href="style.css">
```

## 8. What is DOM?

DOM (Document Object Model) represents HTML as a tree structure. Allows JavaScript to modify content dynamically.

## 9. What are cookies?

Cookies store small data in browser.

**Uses:** 

* Session tracking
* User preferences


<br><br>

SQL is used to manage databases.

**Commands:**
